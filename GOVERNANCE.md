# Governance

DSX Exchange is maintained by NVIDIA.

## Decision Making

Repository maintainers are responsible for:

- Reviewing issues and pull requests.
- Maintaining release and branch hygiene.
- Enforcing project contribution, security, and conduct policies.
- Deciding whether proposed changes fit the project scope.

Technical decisions should be discussed in GitHub issues or pull requests when they affect public behavior, compatibility, security, or contributor workflows.

## Contribution Review

Pull requests require maintainer review before merge. Maintainers may request changes for correctness, maintainability, security, licensing, documentation, or test coverage.

## Releases

Release notes are tracked in [CHANGELOG.md](CHANGELOG.md). Versioning and release automation may vary by component until a repository-wide release process is established.

## Escalation

For public project concerns, open a GitHub issue. For security concerns, follow [SECURITY.md](SECURITY.md). For conduct concerns, contact repository maintainers privately.
