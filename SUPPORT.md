# Support

DSX Exchange is maintained as an open source project with best-effort support.

## Getting Help

Use GitHub issues for:

- Bug reports with reproducible steps.
- Documentation problems.
- Feature requests.
- Questions about local development or validation.

Before opening an issue, search existing issues and check the relevant component README.

## Support Level

Support level: Maintained.

Maintainers review public issues and pull requests as capacity allows. There is no guaranteed response time or service-level agreement.

## Security Issues

Do not report security vulnerabilities through public GitHub issues. Follow [SECURITY.md](SECURITY.md).

## Enterprise Support

This repository does not define a commercial support contract. Use your existing NVIDIA support channel if you have one.
