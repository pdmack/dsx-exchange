# Maintainers

DSX Exchange is maintained by the NVIDIA DSX Exchange maintainers.

## Responsibilities

Maintainers are responsible for:

- Reviewing issues and pull requests.
- Keeping documentation and validation workflows current.
- Maintaining dependency, license, and security hygiene.
- Coordinating releases and repository policy changes.

## Contact

Use GitHub issues and pull requests for public project work. Use [SECURITY.md](SECURITY.md) for vulnerability reports.

Repository-specific maintainer membership is managed through GitHub repository permissions.
